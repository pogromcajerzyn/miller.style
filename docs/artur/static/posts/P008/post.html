In various systems in witch main role is played by waves such as optics or acoustics the main source of trouble can be diffraction. A simple equation is able to predict diffraction pattern but it gets more complicated with ungeometrical shapes. Computer algorithm that I wrote is definitely not the fastest but still accurate enough to show a wave propagation in form of intensity map. The script inputs a boolean transparency mask and calculates a produced image for 2D and 3D case. In case of 3D calculation it is necessary to provide distance from the screen. Max output resolution is 1000x1000px due to matrix size limitations. It is a nice example to show how a simple algorithm can produce accurate and fun outcomes. Point of the simulation is not to make experiments obsolete but to predict outcome to see if the experiment is worth conducting. This is to simple of example to bother with experiments as almost everything that there is to know about diffraction is already discovered. Nevertheless such simulation serves a important purpose: education. For those who have passion of discovery there is nothing more interesting then trying to play with reality that surrounds us and the easiest way to do it is to construct a simulator. After very pleasant and unpredicted by me outcome of this simulation I think this will not be the last simulator that I’ll build. Algorithm uses Huygens’ principle which tells us that every point of a wavefront can be interpreted as an origin of a spherical wavefront. For each transparent point in film and for every pixel program calculates phase difference. Then the second image is produced that is shifted in phase. A standard deviation is used to determine how much each pixel has changed in amplitude when phase of output was shifted. The values of standard deviation are then shown as image.

In example above we can observe that interference occurs when size of slit is bigger then half of wavelength. The light interferes constructively or destructively creating patterns. When we add more slits pattern is getting complicated.
