The formula used to calculate the statistics in "Main power level" graph is a Landers formula (1985): ( 100 * Weight ) / ( 101.3 - ( 2.67123 * repetitions ) ). There are also: The Brzycki (1993), The Baechle (2000) and The Epley (1985) formulas but this one works fine for me.

The fitting of the curve was carried out by using generalised logistic function which in some cases worked out just fine but in some failed as other tried formulas. This can be either due to too poor dataset or the fact that I am not a machine :)

The fitting equation employed is given by: ((a - d) / ((1 + ((x / c) ** b)) ** e)) + d + f * x I took the liberty of adding the linear function to the equation, which resulted in a slight improvement. The grayed-out points are not considered in the fit calculation.
